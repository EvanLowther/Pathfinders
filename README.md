
# Pensylvania Pathways

A pathfinding and graph analysis tool built in C++ that implements and compares Dijkstra’s and A* algorithms on real-world road network data, with CSV export and interactive visualization using Python.



## Installation

In terminal run:
```bash
  pip install cosmograph pandas
```
Now install these two extensions(You can do this in the extension tab as well in vscode)
```bash
code --install-extension ms-python.python
code --install-extension ms-toolsai.jupyter
```

## Running Tests

Build and Run on Mac/Git CodeSpace:

```bash
  ./run.sh
```
Build and Run on Windows
```bash
  ./run.bat
```
